# Knowledge index

This folder holds **distilled** notes, not raw material — and it's not limited to
academic papers. Anything that should inform tuning (papers, articles, blog posts,
vendor docs, conference talks) gets distilled the same way. Raw sources go in
`knowledge/sources/` (gitignored content is fine — PDFs, HTML dumps; for an ordinary web
article, a link in the note is usually enough, no need to mirror the whole page) and get
*distilled* into a short `.md` note under `knowledge/notes/` using the template below.
Nothing gets loaded into an Akamas study or the ROADMAP straight from a raw source — it
passes through a note first.

Use **`/update-knowledge`** (`.claude/commands/update-knowledge.md`) to add a new note —
give it a URL, a pasted excerpt, or a local file, and it handles fetching, distilling
into `notes/_TEMPLATE.md`'s shape, and updating the index below.

Every note uses the same five fields (see `notes/_TEMPLATE.md`):
problem addressed, levers/parameters touched, key results, implications for vLLM/k8s
tuning, which Akamas parameters to explore.
